# Code of Conduct

lazyenv is a small project, but everyone interacting with it — in issues, pull
requests, and discussions — is expected to be respectful and constructive.

The following are not tolerated: harassment, personal attacks, or publishing
others' private information.

Maintainers may edit, remove, or reject contributions and comments that break
these expectations, and may ban repeat offenders from the project.

This applies to all project spaces — the repository, issues, pull requests, and
discussions.

## Reporting

Report unacceptable behavior to the maintainer [@lazynop](https://github.com/lazynop).
